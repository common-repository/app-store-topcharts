A widget to show the top lists of the App Store

== Description ==

App Store TopCharts is a widget that allows you to include all the main classifications of the App Store in the sidebar of your website: 

- Top Free Apps for iPhone
- Top Paig Apps for iPhone
- Top Grossing Apps for iPhone
- Top Free Apps for iPad
- Top Paig Apps for iPad
- Top Grossing Apps for iPad

With App Store Top Charts you can choose the classification you desire,a category of your own choice,the number of elements to show,the store from which you want to extract the classification and the desired theme.


3 themes are included.

With the Geo Location option, you can give the possibility to visitors to see the store's classification of their country.


== Installation ==

1. Upload `app-store-topcharts` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
